# motley-react-redux
Boilerplate for a Motley app with a React/Redux front-end.

## Usage

Install Hipley:
```
$ npm install -g hipley
```

Run the development server (uses config found in `.hipleyrc`):
```
$ hipley --dev
```

Build and run for production:
```
$ hipley
$ motley
```
